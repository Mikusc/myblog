# SceneShift Discussion Room

SceneShift Discussion Room is a mixed-reality coursework prototype for Meta Quest. It reads a real room with Meta Quest scene understanding, applies coherent room stylization themes, and lets the user preview decorative changes to walls, floors, ceilings, windows, and selected furniture.

The prototype is designed for a real discussion room demonstration. It focuses on room decoration and stylization, not on social networking, advertising, or commercial user profiling.

## Main Features

- Mixed-reality room structure loading through Meta Quest room and scene data.
- Theme-based room stylization for room surfaces and selected furniture.
- Optional user style prompts for generating coherent decoration intent.
- Optional online generation pipeline for stylized textures, images, and 3D furniture assets.
- Local fallback assets for stable coursework demonstration.
